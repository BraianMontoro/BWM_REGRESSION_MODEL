# EV Adoption ML

Machine Learning project for predicting electric vehicle adoption across global regions.

## Problem

Electric vehicle adoption varies significantly across markets due to economic factors, fuel prices, and regional trends.

This project builds a machine learning pipeline to predict **high EV adoption** using structured automotive market data.
